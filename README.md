# Hornet Cloak Color

Recolor **Hornet’s cloak** in *Hollow Knight: Silksong* with presets or a custom hex/RGB. Good fit if you want a different look without touching gameplay.

**Multiplayer:** Optional. With **[SSMP](https://thunderstore.io/c/hollow-knight-silksong/p/SSMP/SSMP/)**, players who also have this mod see each other’s cloak colors. SSMP is not required for solo. You can also tint your **floating username** separately (when names are shown); see the same mod’s **Multiplayer Username** section in Mod Menu.

**Requirements:** Silksong, BepInEx, Mod Menu. Keep **`HornetCloakColor.dll`**, **`HornetCloakColor.SSMP.dll`**, and **`cloak_palette.json`** together (e.g. Thunderstore zip layout). Others without the mod still see normal cloaks.
